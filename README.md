# Fundsweb - Security Company Website Template

Fundsweb is a responsive HTML template designed for security companies and service providers. It offers a professional and modern design to showcase security services, expert employees, and company information.

## Features

-   **Responsive Design:** Fully responsive layout that works on all devices.
-   **Service Pages:** Detailed pages for various security services (Office, House, Bank, etc.).
-   **About Us:** specific section to describe the company and its mission.
-   **Contact Form:** Functional contact page with a form.
-   **Blog Section:** To share the latest news and updates.
-   **Testimonials:** Client reviews section.
-   **Gallery/Projects:** Showcase of past projects.

## Technologies Used

-   **HTML5**
-   **CSS3**
-   **JavaScript (jQuery)**
-   **Bootstrap**
-   **Owl Carousel** (for sliders)
-   **FontAwesome** (for icons)

## Folder Structure

-   `assets/css`: Stylesheets (main.css, responsive.css, etc.)
-   `assets/js`: JavaScript files
-   `assets/img`: Images and icons
-   `assets/fonts`: Web fonts
-   `*.html`: HTML pages (index.html, about.html, etc.)

## Customization

To customize the template, you can edit the HTML files for content and `assets/css/main.css` for styling. The layout is built using Bootstrap, making it easy to modify the grid and components.
